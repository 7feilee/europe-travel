# 怎么挑餐馆 · 意大利吃饭规则

> 在意大利吃得好不好，跟花多少钱关系不大，跟你会不会挑关系极大。同一条街上隔两家店，一家是给本地人吃的，一家是给游客拍照的，价差不大，体验天上地下。这一篇给你一套硬核选店法，先讲楼主的三条吃饭秘诀。

## 楼主的三条吃饭秘诀

**① 拒绝"番茄统治一切"的店。** 一进门，菜单从头到尾全是红彤彤的番茄底——番茄意面、番茄披萨、番茄海鲜——这往往是一家把意大利菜简化成游客刻板印象的店。意大利菜的真本事在地方食材、在白汁、在海鲜、在内脏、在时令蔬菜。番茄当然好吃，但**一家只会番茄的店，是偷懒的店**。

**② 去游客少的地方吃。** 这是贯穿全书的原则。景点正门方圆两百米内的餐馆，做的是"这辈子只来一次的人"的生意，没有回头客就没有口碑约束。往居民区走三五个街角，价更低、料更实。

**③ 南意性价比强过北意。** 同样一顿饭，那不勒斯、普利亚、西西里给你的份量、海鲜和热情，北意（米兰、威尼斯）多半给不了。北意贵且端着，南意便宜又生猛。预算有限又想吃好，把重头戏放南意（见 [那不勒斯](那不勒斯与南意-Napoli.md)）。

## 冰淇淋（gelato）：只看桶，不看堆

这条单独拎出来，因为踩坑的人太多。**好 gelato 店和游客陷阱店，看一眼桶就分得清：**

- ✅ **买**：用**不锈钢圆桶装、带盖**、桶口平甚至略低于台面的店。盖着是为了控温防氧化，平桶是因为真材实料不需要堆造型。颜色发暗、不鲜艳（真开心果是暗黄绿、不是荧光绿；真柠檬是奶白、不是亮黄）。
- ❌ **不买**：把冰淇淋**堆得高高的、像小山一样耸起来、五颜六色**摆在敞口柜里招摇的店。那个造型靠的是**植物油打发+色素+乳化剂**，真奶真果做不出那种又高又艳还不化的效果。那是给游客看的橱窗，不是给你吃的东西。

记住：**堆得越高越艳，越是给傻游客准备的。**

## 除了冰淇淋，意大利甜品别抱期望

说句得罪人的实话：**意大利的甜品水平普遍不行。** 大部分城市没有值得你专门跑一趟的甜品店。提拉米苏、cannoli 这些有名归有名，但整体面点和甜品的精细度，跟法国、奥地利差一大截。

唯一的例外是**靠近法国的大城市**——比如**米兰**，受法式影响，甜品和咖啡馆水平能上一个台阶（见 [米兰](米兰与曼托瓦-Milano-Mantova.md)）。除此之外，别把"打卡某网红甜品店"排进行程，多半失望。把胃留给正餐和 gelato。

## 选餐馆的硬核方法

### 一眼就该掉头走的店

- **门口有人站着拉客、招呼你进去、举着带图片的菜单**——掉头。正经馆子不需要拉客；拉客的都是抢游客的店。
- **菜单印着六国语言**（中日韩英德法一应俱全）+ 配大图——这是"我们只做游客生意"的自白。
- **店里坐的全是拖着行李箱、举着相机的人，没有一个本地人**——本地人用脚投票，他们不在的店你也别进。

反过来，**菜单只有意大利语**（顶多附一版英文）、**店里坐着说意大利语的本地人、有老人有小孩**、看起来朴素甚至有点旧——这才是你要找的店。

### 进门前后必问的钱

意大利餐厅的计费规矩跟国内不同，不懂会觉得被宰，其实多是明规则：

- **coperto（座位费/餐具费）**：每人固定几欧，菜单角落或门口会写。这是合法的，不是宰客。但如果高得离谱（一人五六欧以上）就要警惕。
- **servizio（服务费）**：有的店另收 10%–15% 服务费，会写在菜单上。看清楚了再坐。
- **海鲜和牛排按重量计价（all'etto，即每 100 克）**：菜单上写"每 etto 多少钱"而不是整份价。**点之前一定要问清楚这一条大概多重、总共多少钱**——这是意大利沿海最常见的宰客点：你以为点了一条鱼，结账按重量算出三位数。**未标整价的海鲜/牛排，先问价再点，不丢人。**

详细的防宰场景见 [防骗防坑通则](防骗防坑通则.md)。

### 评分要反着用

- **TripAdvisor / Google 评分别正着信。** 一家店**清一色英文好评、又正好开在景点正门**——这恰恰是游客店的特征：游客打分宽松、一次性消费、本地人根本不去。
- 反过来，**评论里意大利语占多数、分数没那么虚高（4.2–4.5 比满屏 4.9 更可信）**的店，往往是本地常客在撑。
- 别迷信"米其林""必比登"标签满天飞——很多是蹭的。

### 尊重意大利的用餐时间

意大利厨房**午后是关门的**。午餐大约到下午两三点收，然后歇到傍晚七点半、八点才开晚餐。你下午四点饿了想找正经馆子吃热饭，基本找不到（只剩 bar 的三明治）。**按当地节奏安排：** 午餐 13:00 前后、晚餐 20:00 之后，才有最好的店开着、最对的氛围。

### 点什么：要本地特色，别点"标准化错菜"

- 点**menu del giorno（今日套餐）**或**当地招牌菜**，这是厨房当天最用心、最当季的部分。
- **别张口就点 "spaghetti bolognese"。** 这道"博洛尼亚肉酱意面"在**博洛尼亚本地根本没有**——本地的是 tagliatelle al ragù（宽面配肉酱），且用的是宽面不是 spaghetti。"spaghetti bolognese"是给外国人发明的菜名，你点它，等于在大声宣布"我没做功课"。
- 千篇一律的 spaghetti、玛格丽特披萨哪儿都有，把胃留给**只有这个地方才有的东西**。

## 按地区，一句话特色

| 地区 | 一句话特色 |
|---|---|
| **皮埃蒙特 Piemonte** | 黄油、白松露、Barolo 红酒，受法国影响、偏厚重精致 |
| **伦巴第 / 米兰** | 米兰烩饭（risotto alla milanese，藏红花）、炸牛排，靠法国所以甜品咖啡也好 |
| **威尼托 / 威尼斯** | 海鲜、墨鱼汁面、cicchetti（小食配酒），但游客区宰得狠 |
| **艾米利亚-罗马涅** | 意大利的胃：帕尔马火腿、帕马森奶酪、tagliatelle al ragù、tortellini，闭眼吃 |
| **托斯卡纳** | 佛罗伦萨大牛排（bistecca alla fiorentina，按重量计价！）、白豆、面包汤，朴实 |
| **罗马 / 拉齐奥** | carbonara、cacio e pepe、amatriciana、犹太区炸朝鲜蓟，羊和内脏 |
| **坎帕尼亚 / 那不勒斯** | 真正的披萨发源地、海鲜、水牛奶酪，性价比之王 |
| **普利亚 Puglia** | orecchiette（猫耳朵面）、生海鲜、橄榄油，淳朴量大 |
| **西西里 Sicilia** | 阿拉伯+希腊底子：海鲜、茄子、cannoli、granita，甜品在意大利里算拿得出手的 |

> 一句话收尾：在意大利吃饭，**会挑的人花小钱吃神仙饭，不会挑的人花大钱吃游客饭**。门口拉客的、菜单六国语言的、冰淇淋堆成山的——这三样看见就走，你已经赢过 90% 的游客了。
