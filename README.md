# 与君王同游 · 硬核欧洲旅游攻略

> 大部分外国游客来欧洲，连机票酒店的成本都没有玩回来，更别说博物馆的票价成本了。
> 这本 wiki 想做的，是让你**玩回你在欧洲花的每一分钱**。

这不是又一份"打卡清单"。中文网络的内容给不了你真正的欧洲，英文网络 90% 是软广告。这里只有三样东西：

1. **做功课的方法** —— 信息源该怎么配比、旅游书该买哪本、行程该怎么从一年前开始排、票该提前多久订；
2. **防坑防骗** —— 假官网、黄牛票、斗兽场焦虑症、番茄统治一切的餐馆，先把坑标出来；
3. **君王做向导** —— 每个目的地，我挑一位最有个性的君王和他的家族，把他们的故事讲给你听。你到了那里，**入其室如见故人**。

> 本系列非常硬核，纯干货，不抒情，偶尔吐槽。如果你想要"人下了飞机才开始看"的保姆式攻略，这里没有。

从北意到南意，以文艺复兴主要城市为核心，逐渐覆盖巴洛克重镇和精品古罗马遗址。详见 [意大利总览与游玩顺序](意大利-总览.md)。

| 大区 | 城市 | 旧政权 | 力荐度 |
|---|---|---|---|
| 皮埃蒙特 | [都灵](都灵-Torino.md) | Savoy 公国 | ⭐ under-rated，初手友好，力推 |
| 威尼托 | [威尼斯 / 维罗纳](威尼斯-Venezia.md) | 威尼斯共和国 | ⭐ 资深游客深度玩 |
| 伦巴第 | [米兰 / 曼托瓦](米兰与曼托瓦-Milano-Mantova.md) | 米兰 / 曼托瓦公国 | 米兰别期望太高；曼托瓦力荐 |
| 艾米利亚-罗马涅 | [博洛尼亚等五城](艾米利亚罗马涅-五城.md) | 教皇国 / 费拉拉 / 摩德纳 | ⭐ under-rated，扒手少，放松 |
| 拉齐奥 | [罗马](罗马-Roma.md) | 教皇国 | 必去，但先治"斗兽场焦虑症" |
| 托斯卡纳 | [佛罗伦萨](佛罗伦萨与托斯卡纳-Firenze.md) | 佛罗伦萨共和国 | 必去，商业化彻底，坑多 |
| 坎帕尼亚 | [那不勒斯及周边](那不勒斯与南意-Napoli.md) | 那不勒斯王国 | ⭐ 楼主最爱，但接受不了乱就别去 |
