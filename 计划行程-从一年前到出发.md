# 计划行程 · 从一年前到出发

> "说走就走"地跑去欧洲热门国家不是潇洒，是小傻；不做准备还赶上旺季去，那就是大傻了。

这一篇是给你照着勾的。把它当 checklist 用，从出发前一年开始，一格一格往下打钩。每条都是楼主吃过亏才总结出来的。

## 时间线总表

| 距出发 | 这阶段必须做完的事 | 勾 |
|---|---|---|
| **约 1 年前** | 开始读历史、逛景点官网、列候选目的地清单 | ☐ |
| **6 个月前** | 确定目的地，做出完整行程计划（落到 Excel） | ☐ |
| **6–12 个月前** | 旺季热门住宿就得开始订（理由见下） | ☐ |
| **3–4 个月前** | 订机票、订酒店、订所有"必须提前订"的门票 | ☐ |
| **出发前** | 查一遍官网，核对开放时间/票价/预约是否有变 | ☐ |

## 第一阶段 · 出发前约 1 年：先看起来

这时候不订任何东西，只做一件事——**让自己配得上这趟旅行**。

- ☐ 开始读目的地的历史脉络（这本 wiki 的国别章 + 旅游书）。
- ☐ 逐个逛景点和城市旅游局的**官网**，记下哪些需要预约。
- ☐ 列一张"候选目的地"清单，先不删减，想去的都写上。
- ☐ 把信息源配比摆正（70% 英文），别一上来就掉进中文网络。具体见 [做功课](做功课-信息源与旅游书.md)。

## 第二阶段 · 出发前 6 个月：把行程钉死

- ☐ 从候选清单里**砍**，定下这一趟真正去的城市和顺序。
- ☐ 排出每座城的天数——宁可少去一城多留一天，别贪。
- ☐ **建一个旅游计划 Excel。** 这是整套行程的指挥中心，没有它你一定会漏订票、撞闭馆。

### 旅游计划 Excel 该有这些列

| 列名 | 填什么 |
|---|---|
| **城市** | 这一段在哪 |
| **天数** | 住几晚 |
| **景点** | 这城里要去的，逐个列 |
| **是否需预约** | 是 / 否——决定它会不会卖完 |
| **预约窗口** | 提前多久放票、官网开抢日期 |
| **营业日** | 周几闭馆、有没有特定休息日（很多博物馆周一闭馆） |
| **餐馆候选** | 提前选好，别到了现场饿着乱找，规则见 [怎么挑餐馆](怎么挑餐馆-意大利吃饭规则.md) |
| **住宿** | 订没订、价位、地址 |
| **交通** | 城际怎么走、要不要租车，见 [交通](交通-自驾公交大巴团.md) |

> 把"营业日"和"是否需预约"两列填实，你就避开了两个最常见的翻车：**兴冲冲跑到门口发现今天闭馆**，和**到了才知道票早卖光只能隔玻璃看**。

## 第三阶段 · 出发前 3–4 个月：开始花钱

到这一步，前面的功课开始变现：

- ☐ **按计划买机票。** 提前三个月左右是较稳的窗口。
- ☐ **网上订热门门票。** 凡是 Excel 里标了"需预约"的，认准**官方域名**下单，别点搜索结果里的广告位（多半是黄牛假官网）。哪些票真会卖完、各自提前多久，全在 [提前订票清单](提前订票清单.md)。
- ☐ 把住宿订完——尤其下面这条。

## 关于住宿：为什么要提前到夸张

订住宿这件事，必须比你以为的更早。给你看个最典型的例子：

> **德国人一年有 6 周带薪假。** 他们提前 **6 到 12 个月**，就把 Garda 湖到 Dolomiti 一带**价廉物美**的住宿全订光了。等你出发前两三个月才动手，剩给你的要么是天价，要么是没人要的烂房。

所以**住宿是整条时间线里最该提前的一环**。热门度假带（湖区、Dolomiti、旺季的海岸城）尤其如此。怎么在一堆房源里挑对，见 [怎么挑住宿](怎么挑住宿.md)。

## 旺季 vs 淡季：怎么取舍

- **旺季**：人多、贵、票紧、餐厅要排——好处是开放最全、白天长。
- **淡季**：人少、便宜、住宿好订、体验从容——代价是部分景点缩短开放或干脆闭门，海岸城可能半睡半醒。

楼主的偏好是**尽量错峰**：同样的钱，淡季能换到安静和余裕，这本身就是体验的一部分。但前提是你得先查清楚——**淡季去之前，务必上官网确认你要的那几个景点没有冬歇闭馆**，这一步就靠你 Excel 里的"营业日"列兜底。

## 最容易翻车的几个时间点

排行程时，下面这几颗雷专门埋在缝里，按勾排查一遍：

- ☐ **周一闭馆**：大量博物馆周一关门，别把重头戏排在周一。
- ☐ **当地公共假日**：复活节、圣诞、各城的主保圣人日，景点和餐馆可能整天歇业。
- ☐ **预约放票日**：有些热门票是"提前 X 天/X 月某时刻整点放"，过了那个点价就翻倍或干脆没了——把这个日期单独设个提醒。
- ☐ **机票和门票的退改条款**：订之前看清楚能不能改，行程一旦微调才不至于全打水漂。
- ☐ **抵达当天别排硬任务**：长途飞行 + 倒时差，第一天留给安顿和踩点，别一下飞机就冲西斯廷。

> 这几条看着琐碎，但每一条都对应着楼主或身边人真摔过的跟头。把它们填进 Excel，比临场反应靠谱一万倍。

> 一句话收尾：**欧洲热门国家不奖励冲动，只奖励提前量。** 把这张表勾完，你就已经赢过机场才打开攻略的那批人了。下一步：[提前订票清单](提前订票清单.md)。
