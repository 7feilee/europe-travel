# 防骗防坑通则

> 欧洲热门城市的骗术不是因为你倒霉才碰上，是**专门冲着游客设计的产业**。提前知道套路，被骗概率直接降一个数量级。

这一篇是全欧通用的防骗防坑总章。具体到某座城特有的坑（比如罗马哪个广场、威尼斯哪条桥），放在各城页里讲；这里只讲**走到哪儿都管用**的那些。看完这篇，再看 [罗马](罗马-Roma.md)、[那不勒斯](那不勒斯与南意-Napoli.md)、[威尼斯](威尼斯-Venezia.md) 的城内细节。

## 一、假官网与黄牛票：本书每个景点都给官网的原因

这是花钱最多、最隐蔽的坑，排第一。

你在 Google 搜一个景点名（比如 "Colosseum ticket"、"Uffizi"），**排在最前面的几个结果，很多不是官网**：

- 一类是**冒充官网**的钓鱼/转售站，页面做得跟官方一模一样；
- 一类是**第三方转售平台**，它把官方票加价转卖，价钱翻倍；
- 更坑的是，加价买来的票**可能还是不可用的**——时段对不上、规则变了、甚至假票，到了门口进不去。

而且搜索结果最上面常是**广告位（Sponsored / 广告）**，这些位置基本被转售平台买断了。

> **认准官方域名。** 意大利官方多是 `.it`，梵蒂冈是 `.va`，法国 `.fr`，西班牙 `.es`，奥地利 `.at`。看清域名再点，**别点广告位**。这就是为什么本书每个重点景点都尽量直接给你官网——就是为了让你绕开这一层。订票整体策略见 [提前订票清单](提前订票清单.md)。

## 二、街头骗术：认套路，不认人

街头骗子的共同点是**主动凑上来、制造一个"你欠了点什么"的情境**。记住套路，见一个躲一个。

- **手链 / 红绳 / "友谊手环"**：常在大教堂、知名广场附近。他笑着上来"送"你、或者直接抓住你手腕往上系绳/编手链，**系上就管你要钱**，不给就纠缠。**手插兜，别伸手，别停步，明确说 No 继续走。**
- **请愿签名党**：一群人（常是少女）拿着夹板让你"为聋哑/慈善签名"。你低头签字的几秒，**同伙在掏你的包**；签完还会指着金额栏要钱。**根本别停下。**
- **街头"免费"赠物**：往你手里塞**鸽食、玫瑰、小饰品**，说是"送你的 / for free"，你一点头一接手，立刻翻脸收费。**手别接任何东西。**
- **假民调 / 假聋哑募捐**：拿张表格假装做调查或募捐，逻辑同签名党，**目的是让你停下、低头、分神**。
- **假警察查钱包 / 验钞**：有人冒充便衣警察，说要"检查你的钱包/护照有没有假钞"。**真警察不会在街上查游客钱包。** 拒绝交出钱包，要求去最近的警察局核实，他们多半自己就溜了。
- **火车站 / 景点"帮你"拎行李、买票**：主动上来帮你提行李、帮你在自动售票机上操作的，**事后伸手要小费**，不给就吵。售票机自己点，行李自己拎。
- **地铁与拥挤处的扒手**：地铁门即将关闭的瞬间一拥而上"挤门"，趁人贴人时下手，**得手就在关门那刻挤出去**，你追不上。人多处把**背包背前面**，手机别插后兜。
- **ATM 贴膜盗刷**：街头独立 ATM 容易被装读卡器和偷拍键盘。**尽量用银行营业厅里面的机器**，取钱时手挡住键盘。

## 三、餐馆坑：宰游客的几种标准操作

吃饭这块的坑，规律性极强。详细的吃饭规则见 [怎么挑餐馆](怎么挑餐馆-意大利吃饭规则.md)，这里只列宰客手法：

- **景点正门口的天价店**：越靠近大景点正门，越贵越难吃，专做"一次性客人"生意。**往里走几条街**再坐下。
- **不标价的海鲜 / 牛排，按 etto 称重宰客**：菜单上海鲜或牛排只写 "S.Q." 或写每 **etto（100 克）** 的价。你点一条鱼，上来按重量一算是天文数字。**点之前一定问清这一份大概多少钱、多重。**
- **不点也上的面包 / 餐前小菜**：服务员"热情"地端上面包、橄榄、小吃，你没点也照样**进账单收费**。不想要，端上来就退回去。
- **coperto 叠加 "service"**：意大利餐桌有 **coperto（座位费/餐具费）**，每人几欧，这是当地规矩、正常。但有些游客区**额外再加一道 "service" 服务费**，等于双重收费。结账**逐项看账单**，看到莫名其妙的项目当场问。

## 四、"斗兽场焦虑症"：别被吓着乱买票

给一个专门的提醒，因为太多人栽在这上面：

你站在斗兽场（或任何热门景点）门口，黄牛和转售网站会拼命制造**"再不买就没了""现在就剩这一档""跟我走免排队"**的紧迫感，把你吓得脑子一热就高价买了不该买的票。

> 这叫**"斗兽场焦虑症"**。它是被人为制造出来卖给你的。真相是：该提前订的票，你按 [提前订票清单](提前订票清单.md) **早就在官网订好了**；没提前订的，现场也有官方售票渠道，犯不着被门口的人牵着走。**深呼吸，认官网，不理黄牛。**

## 五、还有几个"软"坑

这些不偷你钱包，但用道德绑架和信息差掏你：

- **"免费 walking tour" 的小费绑架**：号称免费的徒步团，全程靠**结尾的小费**盈利，导游会用各种方式让你不好意思少给。它不是真免费，是**先服务后绑架你给钱**。想参加可以，心里有数，按自己标准给，别被氛围裹挟。
- **教堂着装与"租披肩"**：进大教堂要求**遮肩遮膝**，这是真规矩。但门口常有人**兜售/出租披肩**，加价不少。**自己带条薄围巾或外套**，几秒钟搞定，省这笔钱。
- **景点门口的假导游**：在景点门口主动凑上来"带你进去/给你讲解"的，很多是**没资质的拉客**，价高、质量差、还可能把你带去转售票点。要导览，走官方或提前订好的正规渠道。

## 落地前 / 落地后 防坑 Checklist

**出发前（落地前）：**

- [ ] 把要去的景点**官方域名**逐个确认、加书签，别现场临时搜。
- [ ] 该提前订的票**全部从官网订好**，存好电子票（见 [提前订票清单](提前订票清单.md)）。
- [ ] 信用卡开通**境外消费提醒**，准备一张额度足够的卡应对租车押金等预授权。
- [ ] 钱财**分散放**：现金、卡分两三处，别全堆在一个钱包里。
- [ ] 护照拍照存云端，记下当地报警/挂失电话。

**到了之后（落地后）：**

- [ ] 背包**背前面**，手机别插后兜，地铁/人群里手按住包。
- [ ] 街上**主动凑上来的一律警惕**：不接东西、不停步、不低头、不交钱包。
- [ ] 取现金用**银行营业厅内的 ATM**，手挡键盘。
- [ ] 吃饭**先看菜单价、问清按重量算的菜、逐项核账单**，远离景点正门口。
- [ ] 出租车**坚持打表**或用正规叫车 App；机场如有官方固定价，认**官方告示牌**的价。
- [ ] 进教堂自带遮肩遮膝衣物；景点门口的"导游/免排队"不理。
- [ ] 门口被催着买票时，想起**"斗兽场焦虑症"**——停一下，认官网，不理黄牛。

> 一句话：**欧洲不危险，但游客会被当成产业来收割。** 你只要不主动停下、不主动伸手、不在门口慌神，绝大多数坑根本碰不到你。
