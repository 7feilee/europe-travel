# 书单与官网清单

> 这一篇是工具页，不是读物。上半部分是楼主信得过的几套大牌旅游书，下半部分是全书提到的、确有把握的景点官网汇总——存下来，订票前对一眼，专治假票黄牛。

旅游书和官网这两样，是 [做功课](做功课-信息源与旅游书.md) 里反复说的"弹药"。书负责"哪里、几点、多少钱、怎么去"，官网负责"真实票价 + 官方订票入口"。这一页把两样都给你列齐。

## A. 旅游书单：一国一册，20 多刀买不了吃亏

一本意大利册，二十几刀或欧，跟一顿正经晚餐差不多的钱，换一整国的底图。**买不了吃亏，买不了上当。** 别舍这点钱，这是旅游里性价比最高的一笔支出。

下面这几套都是欧美书店里站得住脚的大牌，按"风格 + 适合谁"给你点评：

| 书系 | 风格 | 一句话:适合谁 |
|---|---|---|
| **Frommer's** | 实用主义,路线/防坑/预算扎实 | 想要一本"什么都给我安排明白"的省心党。 |
| **Rick Steves**(意大利册及各国分册) | 美式自助游圣经,敢替你砍景点 | 第一次自助欧洲、英文还行、最怕"白跑一趟"的人。**他附自助语音导览 App 和活跃论坛,是最大加分项。** |
| **DK Eyewitness** | 图极多,剖面图/立体图/手绘 | 看字头疼、要先建立画面感的视觉型选手。 |
| **Lonely Planet** | 覆盖最广,背包客底色,小众点全 | 行程往冷门走、想要"连那个小镇都查得到"的人。 |
| **Michelin Green Guide**(绿色,非红色美食册) | 法系,带**评星**(★/★★/★★★)和文化解读 | 想要权威"该看什么、值几颗星"取舍判断的人。 |
| **Blue Guide** | 最学究,硬核历史+艺术,逐件作品讲 | 啃得动的资深玩家、艺术史发烧友。**全场最学术,新手会被劝退。** |
| **Cadogan** | 英式,文笔好、有观点、带性格 | 受不了流水账、想读"有人味、有态度"的旅伴。 |

> 楼主自己的配法:**Rick Steves(管实用和取舍)+ DK 或 Blue Guide(管画面感或硬核艺术)**,两本交叉验证。一本说必去、另一本只字未提,这个落差本身就是信息。具体怎么用这两三本书,见 [做功课](做功课-信息源与旅游书.md)。

### 但这些书都有同一个缺口

不管哪一套,它们都是**按景点逐条描述**的:景点写得细,**建筑和艺术背后的历史只给你简略几句**。原因很简单——它们要把一整国塞进一本书,根本没篇幅展开历史脉络。

所以你会遇到这种情况:书告诉你这座教堂几点开、谁设计的、值几颗星,**却不会告诉你**当年是哪个家族砸钱盖的、和隔壁城为什么世仇、画里那个跪着的人是谁。

> 分工很清楚:**旅游书负责"哪里、几点、多少钱、怎么去",这本 wiki 负责把"为什么、谁干的、跟谁有仇"这条历史脉络补上。** 想把人名家族线串起来,直接看 [君王史玩法](君王史玩法-从初级到资深.md)。

## B. 景点官网清单:认准官方域名,防假票

> **URL 会变,但官方域名后缀通常不变。** 意大利官方多是 `.it`,梵蒂冈是 `.va`,法国 `.fr`,西班牙 `.es`,奥地利 `.at`,德国博物馆常用 `.museum`/`.de`。**看清域名再点,别点 Google 搜索结果顶部的广告位(Sponsored / 广告),也别进任何转售站。** 这就是本书每个重点景点都直接给官网的原因。

下面是全书提到的、楼主有把握的真实官网。订票整体策略见 [提前订票清单](提前订票清单.md);为什么假官网最坑,见 [防骗防坑通则](防骗防坑通则.md)。

### 通用建议:城市旅游局官网

除了上面这些单点官网,**每座城都有自己的官方旅游局网站**(常带 `tourism`/`visit`/`turismo` 字样,挂在本国域名后缀下,如 `turismoroma.it`、`visitvienna` 之类)。它们管三件事:

- **官方活动日历**(临时闭馆、特展、宗教节庆);
- **官方的城市卡/通票入口**(如 Roma Pass 之类,绕开转售);
- **公共交通和实用信息**。

> 同样的规矩:认准本国域名后缀,别在搜索页顶部的广告位里点进去。把这一页和 [提前订票清单](提前订票清单.md) 配着用——一个管"去哪订",一个管"几月开抢"。
