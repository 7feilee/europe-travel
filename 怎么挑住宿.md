# 怎么挑住宿

> 楼主原则上不推荐具体住宿。因为绝大多数发"私藏民宿""神仙酒店"的博主，都是收了好处费的——他们写的是广告，不是经验。所以这一篇我只给你一套自己挑的方法，让你按自己的预算和神经敏感度去定。**没人能替你定该住哪。**

先把话说清楚：楼主自己住过从一百多欧到一千多欧一晚的房间。后者基本不是花现金，是拿酒店点数换的（玩法见下文）。所以如果你在本书某一篇里看到楼主**真的点名推荐了一处具体住宿**，那说明它是真的好——好到我愿意冒"被当成软广"的风险替它背书。除此之外，请你自己挑。

挑住宿不是挑照片，是按下面这八条做减法。

## 一、位置：尽量住进历史中心，但先搞清楚车能不能开进去

意大利、西班牙的老城几乎都是石头路+窄巷，多数核心区是 **ZTL**（Zona a Traffico Limitato，限行区）。这意味着：

- **不自驾的人**：住历史中心步行可达的地方，性价比最高。省下的不是房费，是每天往返的时间和体力。把"晚上看完夜景能走回去"当成硬指标。
- **自驾的人**：你大概率开不进老城，硬开进 ZTL 是**摄像头自动开罚单**，而且是按次累计，回家几个月后账单寄到。要么订带自有车位的城外住宿+白天走进去，要么订老城里"能提供 ZTL 临时通行报备"的酒店（订房时直接问 reception 要不要登记你的车牌）。这条详见 [防骗防坑通则](防骗防坑通则.md) 里关于罚单的部分。

**远离火车站周边。** 火车站附近房子便宜是有原因的：人流杂、扒手和拉客的密度高、夜里不安静。除非你赶早班车、住一晚就走，否则别图那点便宜把整段假期的安全感搭进去。

## 二、噪音：欧洲老城是会响的

照片永远拍不出声音。意大利老城夜里的标配是：街角酒吧到凌晨的人声、清晨的垃圾车、按点敲的**教堂钟**（有的整点+半点都敲，住对面你试试）。订房前盯三件事：

- **临街还是内院**（cortile / courtyard）。内院房安静一个数量级，宁可暗一点。
- **有没有空调**。南意夏天没空调=开窗=把整条街的噪音请进卧室。问清楚是"中央空调"还是"墙上一台老机器"。
- **楼下是不是 bar/餐厅**。订房页面有时不写，去地图街景上看一眼楼下店招。

对噪音敏感的人，宁可贵 30 欧住内院高层，也别赌临街便宜房。

## 三、安全与扒手区识别

订房前在地图上做一件事：看这处住宿离**火车站、大集市、主景点广场**有多近——这三类是扒手高发区。住宿本身在不在这些点的"步行必经路线"上，决定了你每天进出要不要提心吊胆。一楼临街、门禁形同虚设的便宜民宿，行李安全要打个问号。系统的防扒见 [防骗防坑通则](防骗防坑通则.md)。

## 四、早餐：算进总价再比

欧洲酒店报价分"含早 / 不含早"，民宿很多根本没早餐。判断标准很简单：

- **含早是否划算**，要看楼下有没有像样的 bar。在意大利，下楼站着喝一杯 espresso+吃个 cornetto 是几欧的事，比酒店里收你十几二十欧的自助早餐又快又地道。
- 但**早出晚归、赶火车**的行程，含早能省去找地方的麻烦，这时候它值。
- 别被"豪华自助早餐"的照片诱导加钱——你是来逛城的，不是来吃酒店早餐的。

## 五、民宿 / B&B / agriturismo / 酒店 / 连锁积分房，分别适合谁

| 类型 | 适合 | 注意 |
|---|---|---|
| **民宿 / 公寓（apartment）** | 一家人、住 3 晚以上、想自己做点早餐 | 多数无前台，**到店流程要提前问清**（钥匙盒密码 / 房东几点来），晚到航班尤其要确认 |
| **B&B** | 两三晚、想要点本地人情味和早餐 | 房间数少、规矩多（门禁时间），房东好坏决定体验 |
| **agriturismo（农庄）** | 自驾、托斯卡纳/翁布里亚/普利亚乡间、想慢下来 | **必须有车**，晚餐常需预订，地处偏远是它的卖点也是它的门槛 |
| **独立酒店** | 想要前台兜底、随时有人解决问题 | 老城里的小酒店良莠不齐，认评价里的"地段+隔音"两项 |
| **连锁/积分酒店** | 商务感、要稳定品质、攒/用点数 | 老城外居多，看下条 |

## 六、酒店点数玩法：高星用点数换，才划算

楼主住一千多欧的房，靠的不是钱，是**酒店积分体系**。逻辑是：

- 平时出差/刷信用卡攒一个连锁体系（Marriott、Hilton、IHG、Accor 等任选一个深耕）的点数；
- 用点数兑换的，永远挑**现金价最贵的高星房**——同样的点数，换 200 欧的房和换 600 欧的房，点数成本可能差不多，那当然换贵的。
- 旺季、热门城市现金价飙到离谱时，正是点数最值钱的时候。

这套玩法门槛在于你得**提前一两年**就开始攒、并锁定一个体系，不是临时抱佛脚。所以它属于 [计划行程](计划行程-从一年前到出发.md) 的一部分，越早规划越值钱。

## 七、特色住宿：什么时候值得多花钱

楼主反对为照片花钱，但**支持为故事花钱**。值得加预算的，是这几类有真东西的住宿：

- **历史建筑改造**：贵族 palazzo、老修道院（convento / monastero）改的酒店，住进去本身就是逛一处不卖票的古迹；
- **有来历的房子**：作家住过、某段历史发生过的地方；
- 这类住宿的价值在"睡在历史里"，一晚的体验抵得过两张博物馆票。

**但要警惕"网红房"。** 大量房子靠一个超广角、一束打光的角落、一池无边泳池在社媒上爆红，照片之外又小又吵又难到达。识别方法：照片里反复出现的全是同一个网红角落、却看不到房间全貌和窗外实景——就是给你拍照、不是给你住的。订之前找**带实拍、写了缺点**的评价看。

## 八、旺季与热门区域：要么趁早，要么认命

有些区域不是"贵"，是"根本订不到"。最典型的是 **Garda 湖到 Dolomiti 一带**：德国人一年六周带薪假，**提前 6 到 12 个月**就把价廉物美的住宿订光了。轮到你临时去，剩下的不是天价就是烂房。

这种地方只有两条路：**要么趁早订**（半年起步），**要么接受偏高的价格**，没有第三条。把这条记进你的 [计划行程](计划行程-从一年前到出发.md)——订住宿和订机票、订"必抢门票"一样，是越早动手越主动的事。

> 一句话收尾：住宿没有"标准答案"，只有"你的答案"。别人替你定的，往往是替他自己的好处费定的。
