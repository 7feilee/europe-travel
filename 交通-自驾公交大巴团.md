# 交通 · 自驾 / 公交 / 大巴团

> 自驾不是越多越好，火车也不是越纯越好。会玩的人是按**目的地**配交通，而不是按"我爱开车"或者"我怕开车"配交通。

很多人出发前就先把自己绑死在一种模式上：要么"租车自驾才自由"，要么"我不敢开欧洲的车全程火车"。这两种都是初手思维。意大利这种地方，正确做法是**一段一段地拆开看**——这一段火车快又准点就坐火车，那一段景点散在山里没公交就自驾，还有些地方根本不该自己开。本篇讲的就是怎么搭这个 package，以及意大利自驾那一堆能让你回国还收到罚单的坑。

## 楼主的省钱玩法：拿一日游大巴团当交通

先讲个我用了很多次的损招，省钱又省心。

很多分散的景点，公交到不了或者一天倒三趟车，租车又只为这一天不值。这时候去当地旅行社或者酒店前台找**当地一日游大巴团（day tour / escursione）**。它通常是早上某个集合点上车，拉你去 A、B、C 几个点，配个导游讲解，傍晚拉回来。

我的用法是：**买它当交通工具用。** 上了车，到了第一个点下车，跟导游礼貌打个招呼说我自己逛、按时回集合点，然后就甩团自己玩。导游的串讲我不需要——我功课早做完了，比他讲得深。等约定时间回到集合点上车，去下一个点，重复。

> 一趟大巴团的价钱，经常比你为这一天单租一辆车（车租 + 油 + 高速 + 停车 + 还车折腾）还便宜，而且我不用自己开、不用找停车位、不用怕 ZTL。性价比经常**高过租车**。

注意几条：

- 选**纯交通属性强**的团，别选那种全程绑死、必须跟着进购物店/餐厅的"散心团"。
- 上车就跟导游说清楚你要自己活动、几点回集合点，别玩消失让人家等你点名。
- 算好返程车的发车时间，**迟到了车真的会开走**，这是甩团唯一的风险。
- 这招最适合那种"景点之间没公共交通、但有成熟旅游团线路"的地方。

## 自驾不一定更方便——分清场合

"自驾自由"是有前提的。在下面这些情况，自驾反而是负担：

- **大城市市中心**：罗马、佛罗伦萨、那不勒斯、米兰，市中心又堵又贵又限行，开进去纯找罪受，停车费一天能顶半天租车钱。城市间坐火车，城里靠走路 + 地铁。
- **铁路发达的走廊**：北意主干线（米兰—博洛尼亚—佛罗伦萨—罗马—那不勒斯）高铁又快又准点，自驾在它面前毫无优势还更累。
- **威尼斯**：根本不能开车进去，下文专门说。

自驾真正的价值，是在**公交差、景点散**的地方——乡村、酒庄、海岸线、内陆小镇。那里没车寸步难行，有车天高地阔。

## 意大利自驾的坑（楼主重点）

这一节是本篇核心。意大利自驾能踩的坑比你想的多，而且很多是**回国一两个月后**才以罚单形式找上门的。逐条说。

### ZTL：最常见、最贵的坑

**ZTL = Zona a Traffico Limitato，历史中心限行区。** 几乎每座有老城的意大利城市都有，入口立着一块圆牌写着 ZTL，下面常有摄像头。**未经许可的车开进去，摄像头自动拍牌、自动开罚单**，没人当场拦你——你以为没事，其实罚单已经在路上。

- 罚款一次几十欧起，**一段路有好几个摄像头会连罚好几次**。
- 租来的车，罚单先寄到租车公司，租车公司再加一笔"代办手续费"从你信用卡上划走，然后把罚单转给你，**两头挨宰**。
- **导航不一定避得开 ZTL**，便宜导航和某些手机地图照样把你导进去。
- 酒店在 ZTL 里面的，提前问酒店要你的车牌帮你登记白名单（很多老城酒店能办），否则连去自己酒店卸行李都吃罚单。

> ZTL 是托斯卡纳小城的**罚单重灾区**。锡耶纳、圣吉米尼亚诺这种地方，老城基本全是 ZTL，把车停在城墙外的停车场走进去，是唯一不挨罚的做法。

### 租车：自动挡要专门指定

欧洲是**手动挡的天下**。默认租到的就是手动挡。

- 不会开手动挡，**预订时必须明确选 automatic / automatico**，否则到柜台傻眼。
- 自动挡车**更少、更贵**，旺季要更早订。
- 还车记得加满油（满油取、满油还最省心），否则租车公司按高价帮你补油再加服务费。

### 加油：self 和 servito 差价别踩

意大利加油站常有两种价：

- **Self / Fai da te（自助）**：自己刷卡加，便宜。
- **Servito（有人服务）**：有人帮你加，**每升贵一截**，高速服务区尤其黑。

夜里很多小站只剩**无人自助机**，要插钞票或刷卡，**先付款预存再加油**，有些机子认卡挑剔，备点现金。看清价牌再进，别稀里糊涂开到 servito 道上。

### 高速 Autostrada：取卡、还卡、收费

意大利高速 **Autostrada（绿底牌、编号 A 开头）大部分收费**：

- 入口**取通行卡（biglietto）**，出口**插卡按里程付费**，别把卡弄丢。
- 出口认准车道颜色：**白底 Telepass 道是 ETC 专用，没装 Telepass 别进**，进去出不来还得倒车很尴尬；**蓝色刷卡道**和**收现金道**才是你该走的。
- **Telepass** 是意大利的电子收费器，长租或本地人用得多；短途游客一般不办，老老实实走刷卡道。

### 停车线颜色：白蓝黄要分清

路边停车看地上的线，颜色就是规则：

| 线色 | 含义 |
|---|---|
| **白线** | 免费停（但可能有时段/居民牌限制，看牌子） |
| **蓝线** | 收费停，去旁边的**自助缴费机（parcometro）**买票放挡风玻璃下 |
| **黄线** | 居民专用或禁停，**别停**，拖车 + 罚单伺候 |

蓝线区缴费机常只收硬币或刷卡，备点零钱。买的票一定**正面朝上放在仪表台**让查车的看得见。

### 超速：Autovelox 和 Tutor 区间测速

意大利测速很狠，而且是**自动拍照寄罚单**：

- **Autovelox**：固定或移动测速点，拍瞬时速度。
- **Tutor**：**区间测速**，在一段高速的两端各设一个门架，算你这段的**平均速度**——你在中间狂飙再到出口前急刹是没用的，它算平均值。
- 罚单同样会通过租车公司辗转寄到你手上，**半年后收到都正常**。老实按限速开。

### 保险与押金

- 租车默认含基本险，但**自付额（deductible）很高**，刮蹭一下能赔掉一大笔。要不要加买全险看你心脏。
- 取车时**押金会预授权冻结**信用卡一笔钱（可能上千欧），还车没问题才解冻，**备好额度够的信用卡**。
- 取车当场**绕车拍照拍视频**，把已有的划痕全拍下来存证，还车时好自证清白。

### 夜间与南意治安

- 南意和西西里部分地区，**车里别留任何值钱东西，连导航支架都收走**，停车尽量选有人看管的停车场。
- 夜间在不熟的城郊别乱停，导航偶尔会把你带进很窄的老城巷子或很偏的路，**觉得不对就掉头**，别硬开。

## 分地区：到底要不要自驾

把上面这些落到具体地方，给你一张明确的态度表：

| 地区 | 自驾建议 |
|---|---|
| **威尼斯** | **不能开。** 车停 Mestre（陆地端）或岛外停车楼，人坐火车/船进岛。岛上全靠走和坐船。见 [威尼斯](威尼斯-Venezia.md) |
| **[艾米利亚-罗马涅](艾米利亚罗马涅-五城.md)** | **不必自驾。** 铁路极发达，五城之间火车又密又准，全程火车完全替代自驾，还省去 ZTL 烦恼 |
| **[普利亚](普利亚-Puglia.md)** | **必须自驾。** 景点高度分散、公交又差又慢，没车基本玩不动，这是全意大利最该租车的地方之一 |
| **托斯卡纳乡村 / 酒庄** | **适合自驾**，乡间和酒庄没车去不了；但**小城 ZTL 是罚单重灾区**，进城一律停城墙外走进去。见 [佛罗伦萨与托斯卡纳](佛罗伦萨与托斯卡纳-Firenze.md) |
| **南意 / 西西里** | **可自驾但要有心理准备**：路况、驾驶风格、治安、ZTL 一样不少，新手别拿这里练手。见 [那不勒斯与南意](那不勒斯与南意-Napoli.md)、[西西里](西西里-Sicilia.md) |

## 一句话收尾

> 交通是工具不是信仰。能火车就火车，该自驾才自驾，能拿大巴团当交通省一笔就省一笔——**别让一种模式绑架你的行程**。
