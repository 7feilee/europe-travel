# 术语表

> 这是一本随手查的硬核小词典。全书里冒出来的意大利文、艺术史黑话、家族名、菜单暗号,集中放这儿。按"旅游实务 / 艺术建筑 / 历史家族 / 饮食"四类排,每条一句话讲清。

读不懂哪个词,回这儿查;想看家族之间的恩怨脉络,去 [君王史玩法](君王史玩法-从初级到资深.md);菜单上的词怎么用,配 [怎么挑餐馆](怎么挑餐馆-意大利吃饭规则.md);开车坐车的词,配 [交通](交通-自驾公交大巴团.md)。

## 一、旅游实务

| 词条 | 解释 |
|---|---|
| **ZTL**(Zona a Traffico Limitato) | 意大利老城**限行区**。外地车牌开进去会被摄像头自动拍照罚款,**一个路口一张罚单**,几个月后寄到家。自驾的头号坑,详见 [交通](交通-自驾公交大巴团.md)。 |
| **Autostrada** | 意大利**高速公路**,代号 A 打头(如 A1)。多数收费,绿底标志;普通公路是蓝底。 |
| **Autovelox / Tutor** | **测速**设备。Autovelox 是单点固定测速;**Tutor 是区间测速**,算你两点之间的平均车速,踩一脚刹车没用。 |
| **coperto** | **餐位费 / 桌布费**,意大利餐馆按人头收的固定费用(常见 1.5–3 欧/人),账单底部出现属正常,不是宰你。 |
| **servizio / cover charge** | **服务费**。`servizio` 是意式服务费;英文菜单上写 cover charge 多指 coperto。两者通常二选一,**别和小费混为一谈**。 |
| **aperitivo** | **餐前酒**时段(傍晚约 6–8 点)。点一杯酒,常**免费附赠小食**,北意(米兰、都灵)尤其讲究,是高性价比的轻晚餐。 |
| **agriturismo** | **农庄旅宿**。乡间农场改的住宿+餐饮,吃自家产的食材,托斯卡纳、普利亚常见,体验比城里酒店地道。 |
| **acqua alta** | 威尼斯的**高水位**现象,秋冬季海水倒灌淹没广场,临时架高架步道(passerella)通行。 |
| **Telepass** | 意大利高速的**电子不停车收费**装置/车道。租车若没配,**别误入 Telepass 专用道**,会卡住或罚款。 |
| **all'etto**(per etto) | **按 100 克计价**。海鲜、牛排菜单上常只标"每 etto 多少钱",看着便宜,一称重就是天价。**点之前务必问清这一份大概多重、多少钱。** |

## 二、艺术建筑

| 词条 | 解释 |
|---|---|
| **fresco**(湿壁画) | 趁灰泥**未干**时直接上色,颜料渗进墙体、与墙同寿。米开朗琪罗的西斯廷顶就是。干了再画的叫 secco,容易脱落。 |
| **Quattrocento / Cinquecento** | 字面"四百 / 五百",指**1400 年代(15世纪)/ 1500 年代(16世纪)**。Quattrocento 是早期文艺复兴,Cinquecento 是盛期。 |
| **Mannerism**(手法主义) | 盛期文艺复兴**之后、巴洛克之前**的过渡风格。人物刻意拉长、姿态扭曲、构图不稳,是对"太完美"的反叛。 |
| **Baroque**(巴洛克) | 17 世纪起的**戏剧化**风格:强烈明暗、动感、宗教激情,贝尔尼尼、卡拉瓦乔是代表。天主教反宗教改革的视觉武器。 |
| **Rococo**(洛可可) | 巴洛克的**轻甜化**晚期版本,18 世纪,粉嫩、繁复、贵族沙龙趣味,常见于美泉宫一类宫殿内饰。 |
| **contrapposto**(对偶倒立式) | 雕像把**重心压在一条腿**上,胯和肩反向倾斜,身体呈 S 形,显得自然、有生命。大卫像的标准站姿。 |
| **pietà**(圣殇/哀悼基督) | 圣母**怀抱死去基督**的题材。最有名的是梵蒂冈那尊米开朗琪罗的《圣母怜子》。 |
| **trompe l'oeil / di sotto in sù**(仰角错觉) | `trompe l'oeil` 是**以假乱真的错视画**;`di sotto in sù`(由下往上看)是天花板上画的**仰角透视**,让你站底下感觉穹顶通天。曼托瓦的婚礼堂是教科书。 |
| **console**(牛腿/托座) | 建筑上撑出檐口、阳台或壁炉架的**涡卷形支托**,常成对出现,巴洛克立面上极常见。 |
| **Flemish Primitives**(早期尼德兰画派) | 15 世纪佛兰德斯的**油画先驱**(凡·艾克等),以惊人的细节和油彩光泽著称,比意大利更早玩透油画。 |
| **mosaic**(拜占庭马赛克) | 用**金箔玻璃和石片**镶嵌的画,拜占庭风格满墙金光,威尼斯圣马可、拉文纳、西西里蒙雷阿莱是巅峰。 |
| **Grand Tour**(壮游) | 17–19 世纪英国贵族子弟的**意大利游学传统**,以罗马为核心补古典教养。歌德的意大利游记就是其中一例,本书的玩法正是它的现代版。 |

## 三、历史 / 家族 / 政体

| 词条 | 解释 |
|---|---|
| **Doge** | 威尼斯共和国的**终身选举元首**,有名无实权、受重重制约,是寡头共和的门面。 |
| **Serenissima**(至尊共和国) | 威尼斯共和国的**自称**,"最尊贵的"。存续上千年,直到拿破仑才终结。 |
| **Signoria** | 意大利城邦的**统治机构/政权**形态,常指佛罗伦萨的市政府,也泛指由某强人/家族把持的"僭主统治"。 |
| **condottiero**(雇佣兵首领) | 文艺复兴意大利的**雇佣兵团头目**,替城邦打仗,势力大了常自己上位当主公(如斯福尔扎)。 |
| **Medici**(美第奇) | **佛罗伦萨**的银行业家族,赞助文艺复兴、出过教皇和王后,实际统治托斯卡纳数百年。 |
| **Este**(埃斯特) | **费拉拉 / 摩德纳**的统治家族,宫廷文化极盛。 |
| **Gonzaga**(贡扎加) | **曼托瓦**的统治家族,曼特尼亚、鲁本斯都为其服务,婚礼堂出自其手。 |
| **Sforza / Visconti**(斯福尔扎 / 维斯孔蒂) | **米兰**的两代统治家族:维斯孔蒂在前,雇佣兵出身的斯福尔扎接手在后,达·芬奇的赞助人即此族。 |
| **Borgia**(波吉亚) | 西班牙裔家族,出了**两位教皇**(含亚历山大六世)和狠角色切萨雷·波吉亚,文艺复兴权谋与丑闻的代名词。 |
| **Farnese**(法尔内塞) | 由教皇保罗三世发迹的家族,据有**帕尔马**,留下罗马法尔内塞宫和那不勒斯的法尔内塞收藏。 |
| **Savoy**(萨伏依) | 起于**都灵/皮埃蒙特**的王朝,最终**统一意大利**、成为意大利王室,直到 1946 年共和。 |
| **Habsburg**(哈布斯堡) | 称霸中欧数百年的王朝,核心在**维也纳**,版图一度横跨欧洲(及西班牙分支),美泉宫、KHM 都是其家底。 |
| **Aragon**(阿拉贡) | 西班牙东部王国,中世纪**统治过那不勒斯、西西里、撒丁**,把伊比利亚势力带进南意。 |
| **Bourbon**(波旁) | 法国王室,其一支为**两西西里王国**(那不勒斯+西西里)的国王,卡塞塔王宫即波旁所建。 |
| **Hohenstaufen**(霍亨斯陶芬) | 中世纪德意志王朝,其腓特烈二世把统治中心放到**西西里**,留下普利亚的城堡群。 |
| **Papal States**(教皇国) | 教皇直接统治的**中意大利**世俗领地,横亘千年,1870 年才被并入统一的意大利。 |
| **stupor mundi**("世界奇迹") | 对**腓特烈二世**(霍亨斯陶芬)的称号,通晓多语、好科学、与教廷死磕,普利亚那座几何完美的 Castel del Monte 是他的纪念碑。 |
| **Mezzogiorno**(南意) | 字面"正午",指意大利**南部**(那不勒斯以南、西西里等),历史、经济、风土自成一系,和北意是两个世界。 |

## 四、饮食

| 词条 | 解释 |
|---|---|
| **ragù**(肉酱) | 意式**慢炖肉酱**。博洛尼亚的正宗版配宽面(tagliatelle),**根本没有"spaghetti bolognese"这道菜**——那是给外国人编的,本地人不认。 |
| **gelato**(意式冰淇淋) | 脂肪低、密度高、现做。**鉴别真假看桶**:盖**金属/不锈钢桶**、颜色暗淡平摊的多为真材料;堆得老高、颜色艳丽(尤其荧光绿开心果)的多是香精工业货,绕道。 |
| **DOP / IGP** | 欧盟**原产地保护**认证。DOP 最严(产地+工艺全锁定),IGP 略松,看到这标志基本是地道货。 |
| **burrata** | 一团**外皮是马苏里拉、内芯是奶油**的鲜奶酪,一切开流心,普利亚特产,**极不耐放,要趁鲜吃**。 |
| **Parmigiano-Reggiano**(帕马森干酪) | 受 DOP 保护的**真·帕马森**,只产自指定产区。超市那种粉状"Parmesan"是仿货,不是一回事。 |
| **balsamico**(香醋) | 摩德纳的**葡萄醋**。真正的 Tradizionale 陈酿数十年、浓稠如糖浆、按小瓶卖,和超市几欧一瓶的工业调味醋天差地别。 |
| **cucina povera**("穷人的菜") | 意大利的**贫民料理传统**:用最便宜的边角料(豆子、面包、内脏、野菜)做出极好吃的东西,南意精髓,常比大鱼大肉更见功夫。 |
| **Margherita**(玛格丽特披萨) | 那不勒斯的**基准披萨**:番茄红、马苏里拉白、罗勒绿——意大利国旗三色。一家披萨店行不行,点这一张最见真章。 |
