# 做功课 · 信息源与旅游书

> 你 90% 看中文网络、10% 看英文黄牛票网站，那你玩不好是注定的——这跟你聪不聪明、花多少钱都没关系，是源头错了。

做功课不是玄学，是有正确配比、正确顺序、正确工具的一门手艺。这一篇把楼主用了多年的那套摊开给你。

## 信息源的正确配比

记住这个比例，它是这一整套攻略的地基：

| 语言 | 占比 | 你在这里查什么 |
|---|---|---|
| **英文** | **70%** | 大品牌旅游书、专业攻略、英文官网、TripAdvisor / Rick Steves 论坛 / reddit / facebook group |
| **当地欧洲语言** | **10–20%** | 当地旅游局官网、博物馆官方页、本地餐馆的真实评价 |
| **中文** | **10–20%** | 极少数真正懂的人写的长文、突发信息、临时找住宿 |

再看看大多数游客的实际配比：**90% 中文网络 + 10% 英文黄牛票网站。** 这就是为什么绝大部分游客来欧洲，**连机票酒店的成本都玩不回来**，更别提门票钱了。不是欧洲坑，是他们的信息源从根上就反了——拿着错的地图，走得越勤，离得越远。

> 配比这事没有商量余地。你可以语言不好、可以借助翻译，但你的信息**重心**必须压在英文上。把重心放在中文网络，就等于自愿把自己降级成被宰的散客。

## 做功课需要哪几样东西

凑齐这四样，你就有了一套完整的弹药：

1. **两到三本大品牌旅游书。** 一国一册，详见下一节。
2. **各景点和城市旅游局的官网。** 官网管两件事：真实的开放时间/票价，以及**官方订票入口**（绕开假官网和黄牛）。汇总见 [书单与官网清单](书单与官网清单.md)。
3. **真正懂旅游的人写的网页。** 不是流量号，是那种一篇文章写五千字、带门票链接、敢下判断的人。这种人全网就那么些，找到了就长期跟着。
4. **论坛和社群**：TripAdvisor、Rick Steves 论坛、相关 subreddit、对口的 facebook group。这里有海量"我刚去过、实际情况是这样"的一手信息，是旅游书更新不及的部分。

## 小某书的唯一真实用处

我不是无脑黑。小某书确实有两个 Google 替代不了的真实场景：

- **突发信息。** 比如你正走在去梵蒂冈的路上，它突然起火关闭了——这种几小时内发生、官网还没更新的事，社媒上的实时帖比任何旅游书都快。
- **临时找住宿。** 比如华人黑旅馆这类不上正规平台的房源。**（楼主不推荐，但承认它能解燃眉之急。）**

除了这两类，**小某书能给你的，Google 全都有，而且更准更全。** 别把"实时性"和"权威性"搞混了：它实时，不代表它对。

## 旅游书：20 多块钱，买不了吃亏

一本意大利册旅游书，二十几刀或欧。**买不了吃亏，买不了上当。** 跟你一顿正经晚餐差不多的钱，能换一整国的底图，这是旅游里性价比最高的一笔支出，没有之一。

- **Frommer's / Rick Steves**：非常有用，路线、防坑、实用信息扎实，Rick Steves 尤其擅长帮你"砍掉不值得去的"。
- **DK / Blue Guide**：DK 图多直观、适合建立画面感；Blue Guide 偏学术、适合啃硬核艺术史。

**这些书共同的缺点：** 它们是**按景点逐条描述**的，建筑和艺术相关的历史只给你**简略几句**——它们要塞进一整国，没篇幅展开脉络。

> 所以分工很清楚：**旅游书负责"哪里、几点、多少钱、怎么去"，这本 wiki 负责把"为什么、谁干的、跟谁有仇"这条历史脉络给你补上。** 两边合起来，你才既不踩坑，又看得懂。

### 怎么用这两三本书

买回来不是供着的。楼主的实际用法：

- **第一遍通读对应国别册的总论部分**，建立"这国大概几个区、各区什么底子、我该去哪几个"的全局判断——很多人跳过总论直奔景点，是丢了西瓜捡芝麻。
- **景点条目当词典查，不当小说读。** 排到哪城就翻哪城，把要去的逐条标出来，和你的 Excel 对上。
- **特别看作者下的"评级"和"取舍建议"。** Rick Steves 会明说哪个景点"不值得专门绕路"，这种敢替你砍的判断，比满屏好评有用得多。
- **两本交叉验证。** 一本说必去、另一本只字未提，这个落差本身就是信息：说明它要么小众、要么被高估，正好上论坛再查一手。

## 正确的做功课顺序：第一遍别看视频

做第一遍功课，尽量别碰视频。原因有三：**太费时间**、**创作者水平参差还普遍偏低**、**错误一堆还以讹传讹**。一个十分钟视频能讲的干货，文字三十秒就读完了。

正确顺序是：

1. **先啃专业旅游书**——建立全局框架。
2. **再搜文字攻略**——补细节、查最新情况。
3. **最后**，当你卡在"这个景点到底值不值得专门跑一趟"的时候，**才去看视频**——这时候你已经有判断力了，视频只是帮你做最后那一下取舍。

把视频放在最后，而不是最前，是新手和老手最容易拉开差距的一步。
